Administration Overview

Embedthis Appweb is administered via an Apache style configuration file: appweb.conf. This configuration file takes directives which control the operation of Appweb. When running, Appweb logs its configuration, errors and events of interest to its error log which is specified in the configuration file, but which usually defaults to <InstallDir>/logs/error.log.

 The Administration Guide explains the following topics:

  • Starting and Stopping Appweb

  • Running as a Service

  • Security

  • Log Files
